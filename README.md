# IoT Technical Survey Series

This repository contains a structured collection of technical survey reports
documenting my self‑study journey in Internet of Things (IoT).

The goal of this series is to build a solid **system‑level understanding of IoT**
—from fundamentals to hardware, software, cloud, security, and real‑world
applications—before moving to implementation and projects.

---

## 🎯 Phase 1: IoT Foundation

A strong conceptual and architectural foundation of IoT systems.

- **IoT_01** – Introduction to Internet of Things ✅
- **IoT_02** – Communication Protocols in IoT (MQTT, HTTP, CoAP, LoRaWAN, BLE)
- **IoT_03** – Sensors in IoT Systems (Types, Analog vs Digital, Calibration)

---

## 🎯 Phase 2: Hardware Track

Understanding the physical layer of IoT systems.

- **IoT_04** – Microcontrollers & Embedded Systems for IoT  
  (ESP32, STM32, Arduino, Peripherals)
- **IoT_05** – Power Management in IoT Devices  
  (Batteries, Deep Sleep, Energy Optimization, Solar IoT)

---

## 🎯 Phase 3: Software Track

Embedded software fundamentals and architecture.

- **IoT_06** – Embedded C Programming Basics for IoT Systems
- **IoT_07** – Real‑Time Operating Systems in IoT (FreeRTOS Basics)
- **IoT_08** – Software Architecture for IoT Devices  
  (HAL, Drivers, FSM, Layered Design)

---

## 🎯 Phase 4: Cloud, Security & Applications

End‑to‑end IoT system design.

- **IoT_09** – IoT Cloud Platforms  
  (Firebase, AWS IoT, ThingsBoard, Node‑RED)
- **IoT_10** – IoT Security  
  (Attacks, Encryption, Secure Communication, OTA)
- **IoT_11** – IoT Applications  
  (Smart Home, Smart Agriculture, Smart Factory, Healthcare, Smart Cities)
- **IoT_12** – Edge Computing vs Cloud Computing in IoT

---

This repository serves as a **technical documentation portfolio** and a learning
reference that focuses on architecture, design decisions, and engineering
trade‑offs rather than isolated tools or tutorials.
