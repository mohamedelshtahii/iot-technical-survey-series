## IoT_02 – Communication Protocols in Internet of Things

This technical survey report covers the most commonly used communication
protocols in IoT systems, focusing on system-level understanding and
engineering trade-offs.

### Topics Covered
- Role of communication in IoT architecture
- MQTT protocol and publish/subscribe model
- HTTP usage in IoT systems
- CoAP for constrained devices
- BLE for short-range IoT communication
- LoRaWAN for long-range low-power networks
- Comparison between IoT communication protocols

This report is part of the **IoT Technical Survey Series**.
